Choose a problem to solve

Some info to 

  • How to select a project to solve
    - Industry to choose from (Might be provided to you)
    - Type of problem Supervised/Unsupervised, Classification/regression (Will be decided by the dataset that you choose)

  • Conduct Literature Review. Read up on how others have solved this or similar problems 

  • Project feasibility: Do you have enough data? Do you have the tools to process the data

  • Read through resources to help design a strategy to attack the problem

  • What will you report in the end? Hypothesis?
